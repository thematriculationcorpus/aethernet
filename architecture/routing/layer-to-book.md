# Layer → Book Visibility Matrix (Read-Only)

This document defines which Books may observe which Layers.
No execution. No authority escalation.

## Visibility Rules

- VAULT → ACCORDANCE (exclusive)
- DOOR → COMPLIANCE
- WINDOW → COMPLIANCE
- HABITAT → AVOIDANCE
- MONOCLE → All Books (observational only)
- AETHIE → None (HUD only)
- AETHERNET → None (transport only)
- AETHER → None (consolidation only)

## Absolute Prohibitions

- COMPLIANCE may not observe VAULT
- AVOIDANCE may not observe VAULT
- WINDOW may not access ACCORDANCE
