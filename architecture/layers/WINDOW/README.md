# WINDOW

## Role
WINDOW is the public-facing interface layer.
It presents services, outputs, and representations to external audiences without exposing internal structure.

## Position in Architecture
WINDOW sits downstream of DOOR and upstream of HABITAT.
It is the lowest-privilege exposure layer.

## Inputs
- Structured outputs from DOOR
- Approved public representations
- Presentation-ready artifacts

## Outputs
- Public-facing services
- Displayed information
- Customer-visible interactions

## Non-Responsibilities
- No execution
- No decision-making
- No governance
- No interpretation
- No access to VAULT
- No back-channel communication
