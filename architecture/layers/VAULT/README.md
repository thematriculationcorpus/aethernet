# VAULT

## Role
VAULT is the private core containment layer.
It holds protected structures, doctrines, records, and authorities that are not public-facing.

## Position in Architecture
VAULT sits upstream of DOOR and WINDOW and downstream of AETHER.
It is never directly exposed.

## Inputs
- Canonical records
- Private doctrines
- Governance artifacts
- Protected state snapshots

## Outputs
- None directly

## Non-Responsibilities
- No execution
- No communication to public layers
- No automation
- No interpretation
- No persuasion
- No external interface
